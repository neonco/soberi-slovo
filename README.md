# 🎯 Логопед — Собери слово

**Последнее обновление:** 18.07.2026
**Цель:** Приложение для логопедических занятий с ребёнком 5 лет (ОНР 2, стереотипная речь)

> Работаешь кодовым агентом? Сначала прочти **[AGENTS.md](AGENTS.md)** — там правила сборки и архитектура.

---

## 🚀 Запуск

```bash
cd prototype
npm run dev        # → http://localhost:7100
```
Или просто открыть `prototype/index.html` двойным кликом.

## 📁 Структура

```
prototype/   ← приложение (шаблон + собранный index.html + сервер)
content/     ← базы слов: 143 слова по 7 уровням Марковой + цифры/растения/профессии/глаголы
tools/       ← build.py (сборка), gen_images.py, stt_test.py, syllable_text.json
samples/     ← записи голоса ребёнка для теста распознавания
docs/        ← research.md (методика), ARCHITECTURE.md
```

## 🎮 Режимы

| Режим | Что делает |
|---|---|
| 🧩 **Собери слово** | 7 уровней слоговой структуры по Марковой. Тап по слогу → в слот; тап по слоту — возврат; автопроверка после последнего слога; цвета слогов как на бумажных карточках |
| 👂 **Покажи!** | Звучит «покажи …», 4 карточки — выбрать правильную. 14 категорий |
| 🏃 **Что делает?** | Тренажёр 12 базовых глаголов |
| 💬 **Да или нет?** | «Это киса?» — 👍/👎 |

Настройки для взрослых: диктор (7 голосов Яндекс SpeechKit, mp3-файлы), темп ползунком, эмодзи/карточки (185 настоящих карточек), перемешивание слогов и карточек, подсветка ударения, переход к любой карточке уровня (кнопка «2/23 ▾»), запись образцов речи для STT-теста.

Прогресс (освоенные слова) хранится локально в браузере.

## 🔧 Изменение приложения

```bash
# 1. Править prototype/index.template.html и/или content/*.json
# 2. Собрать:
python tools/build.py
```

Новые слова → догенерировать озвучку (7 дикторов, см. AGENTS.md) и картинки (`tools/gen_images.py`).

## 🧠 Контекст ребёнка

- Возраст 5 лет, ОНР 2 (стереотипная речь), активный словарь ~6 слов, нормальный слух
- Подход: расширение слоговой структуры (Маркова), переход от эхолалии к спонтанной речи, без давления на «правильность» — ошибка это мягкая тряска и «попробуй ещё»
