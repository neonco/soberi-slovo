## Прогресс дочери

| Слово | Слоги | Уровень | Статус |
|-------|-------|---------|--------|
| **мама** | ма-ма | 1 (CV-CV) | ✅ Освоено |
| **папа** | па-па | 1 (CV-CV) | ✅ Освоено |
| **баба** | ба-ба | 1 (CV-CV) | ✅ Освоено |
| **вода** | во-да | 1 (CV-CV) | ✅ Освоено |

**Интересно**: банан (ба-нан) — это уже **уровень 4** по Марковой! Значит, закрытые слоги (конечный согласный) она тоже может — просто не осознаёт это как отдельный навык. Это хороший знак для дальнейшего прогресса.

## Уровни сложности (Маркова)

```
УРОВЕНЬ 1 — ТЕКУЩИЙ (освоено 6/23)
├── Структура: CV-CV (открытые слоги)
├── Примеры: ма-ма, во-да, ки-са, мя-со, но-га, ру-ка
└── Следующие цели: коза, сова, змея, дядя, тётя, ляля

УРОВЕНЬ 2 — СЛЕДУЮЩИЙ ШАГ
├── Структура: CV-CV-CV (три открытых)
├── Примеры: мо-ло-ко, ма-ши-на, со-ба-ка, ко-ро-ва
└── Фокус: удлинение, сохраняя открытость

УРОВЕНЬ 3 — ПАРАЛЛЕЛЬНО (закрытые слоги)
├── Структура: CVC (один слог)
├── Примеры: кот, дом, сок, нос, мяч
└── Ключевой момент: банан уже показывает, что она УМЕЕТ конечный согласный

УРОВЕНЬ 4 — УЖЕ НАЧАТ (банан)
├── Структура: CV-CVC
├── Примеры: ба-нан, ди-ван, ко-тик, пла-ток
└── Цель: осознанное использование закрытого слога

УРОВЕНЬ 5 — В ПЛАНЕ
├── Структура: CV-CCV (стечение согласных)
├── Примеры: бан-ка, вет-ка, кош-ка, лам-па
└── Трудность: два согласных подряд

УРОВЕНЬ 6 — В ПЛАНЕ
├── Структура: CVC-CVC (закрытые+закрытые)
├── Примеры: ком-пот, сал-ат, то-пол
└── Трудность: согласный на стыке слогов

УРОВЕНЬ 7 — В ПЛАНЕ
├── Структура: CV-CV-CVC
├── Примеры: бе-ге-мот, те-ле-фон, са-мо-лёт
└── Самый сложный из освоенных типов
```

## Как использовать базу

1. **Выбирайте категорию** по интересу ребёнка (животные/еда/семья)
2. **Начинайте с освоенных** — повторение для уверенности
3. **Добавляйте 1-2 новых слова** из уровня 1 за сессию
4. **Пробуйте уровень 2-3** — если получается, отмечайте в JSON
5. **Не торопитесь** — стереотипная речь требует времени на расширение

---

## Формат JSON

Каждое слово содержит:
- `word` — слово целиком
- `syllables` — массив слогов
- `stress` — индекс ударного слога (0-based)
- `image` — ключ для поиска картинки
- `category` — категория
- `status` — mastered / learning / planned
