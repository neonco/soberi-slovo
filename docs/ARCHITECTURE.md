# Архитектура приложения «Логопед»

## Стек

| Компонент | Технология | Почему |
|-----------|-----------|--------|
| **Фреймворк** | Flutter (Dart) | Один код → Android, iOS, Web, Desktop |
| **Игровой движок** | Flame (на Flutter) | 2D-игры, анимации, жесты — идеально для детских карточек |
| **Хранение** | Hive (local) + Firebase (опционально) | Оффлайн-режим обязателен |
| **Аудио** | audioplayers (Flutter) | Воспроизведение слогов, музыки |
| **TTS** | flutter_tts | Озвучивание слов (оффлайн-голоса) |
| **Распознавание речи** | speech_to_text (MVP) + Vosk (продакшн) | Проверка произношения |
| **Сборка** | Flutter build | `flutter build apk` / `flutter build ios` / `flutter build web` |

## Модули

```
lib/
├── main.dart              # Точка входа
├── models/
│   ├── word.dart          # Модель слова (из syllable-cards.json)
│   ├── level.dart         # Модель уровня
│   └── progress.dart      # Прогресс ребёнка
├── data/
│   └── word_repository.dart # Загрузка слов, фильтрация
├── games/
│   ├── syllable_builder.dart  # «Собери слово» — конструктор слогов
│   ├── rhythm_tap.dart        # «Хлопни в слог» — ритм
│   ├── echo_repeat.dart       # «Повтори за мной» — микрофон
│   ├── mand_game.dart         # «Что хочет [персонаж]?» — запрос
│   ├── find_pair.dart         # «Найди пару» — фонематика
│   └── articulation.dart      # «Артикуляционная зарядка»
├── ui/
│   ├── home_screen.dart       # Главный экран (выбор игры)
│   ├── level_select.dart      # Выбор уровня
│   ├── parent_dashboard.dart  # Родительский дашборд
│   └── settings.dart          # Настройки
├── services/
│   ├── audio_service.dart     # Звуки, TTS, музыка
│   ├── speech_recognizer.dart # Распознавание речи
│   └── progress_service.dart  # Сохранение/загрузка прогресса
└── utils/
    ├── constants.dart         # Константы, цвета
    └── validators.dart        # Валидация произношения
```

## Первый релиз (MVP)

**Фичи:**
1. Одна мини-игра «Собери слово»
2. Уровень 1 (CV-CV) — 10 слов
3. TTS озвучивание слогов
4. Анимация успеха (звёздочки + звук)
5. Оффлайн (все данные в приложении)
6. Родительский дашборд — прогресс по словам

**Следующие релизы:**
- v0.2: Уровни 2-3 + игра «Хлопни в слог»
- v0.3: Микрофон + базовое распознавание
- v0.4: 6 мини-игр + родительские настройки
- v1.0: Полная база 140 слов + экспорт статистики

## Сборка для Rustore

```bash
# Android (APK/AAB)
flutter build apk --release
flutter build appbundle --release  # для Rustore

# iOS (требуется Mac + Xcode)
flutter build ios --release

# Web (для демо)
flutter build web --release
```

Rustore принимает AAB (Android App Bundle) — как Google Play.
