# Исследование: приложение для речевых тренировок детей с ЗПРР/РАС

> Статус: сбор данных завершён, стадия систематизации
> Дата: 2026-07-18
> Целевая аудитория: ребёнок 5 лет, ЗПРР, возможный РАС, уровень: слоговое произношение (во-да, ки-са, ма-ма)

---

## 1. Обзор научных подходов (Evidence-Based)

### 1.1 ABA (Applied Behavior Analysis) — Прикладной анализ поведения
- **Статус**: наиболее изученный подход, признан US Surgeon General и National Academy of Sciences
- **Meta-analysis 2025**: 25 RCT, значимый эффект на адаптивное поведение, бытовые навыки, язык
- **Ключевой принцип**: разбиение навыков на мелкие шаги + позитивное подкрепление
- **Интенсивность**: 20-40 часов/неделя для малышей, чем раньше — тем лучше

**Verbal Behavior Therapy (VBT)** — подвид ABA, разработанный на основе теории Скиннера:
- **Mand** — запрос (самый важный для начала!)
- **Tact** — называние/комментирование
- **Echoic** — повторение звуков/слов
- **Intraverbal** — ответы на вопросы
- **VB-MAPP** — стандартная оценка прогресса

### 1.2 NDBI (Naturalistic Developmental Behavioral Interventions)
Объединяют поведенческие и развивающие подходы:
- **Early Start Denver Model (ESDM)**
- **Pivotal Response Treatment (PRT)**
- **JASPER** (Joint Attention, Symbolic Play, Engagement, Regulation)
- **Enhanced Milieu Teaching (EMT)**

**Ключевые элементы**:
- Обучение в естественной среде (дом, игра)
- Совместный контроль ребёнок-взрослый
- Следование за интересами ребёнка
- Моделирование, shaping, prompting, дифференциальное подкрепление
- Целевые навыки: имитация, совместное внимание (joint attention), очередность, предречевые жесты

**Meta-analysis 2021 (Sandbank et al.)**: положительные эффекты на социальную коммуникацию, язык, игру, когнитивные навыки.

### 1.3 AAC (Augmentative and Alternative Communication) — Альтернативная коммуникация
Критически важно для невербальных/минимально вербальных детей:

**PECS (Picture Exchange Communication System)** — 6 фаз:
1. Физический обмен карточкой
2. Дистанция и настойчивость
3. Дискриминация картинок
4. Построение предложений ("Я хочу...")
5. Ответ на "Что ты хочешь?"
6. Комментирование

**Исследования**:
- Charlop-Christy et al. (2002): 3 ребёнка с РАС — освоили PECS + параллельный рост речи
- Meta-analysis (Flippin et al., 2010): малый/умеренный эффект на коммуникацию, речь — малый/отрицательный
- **Важно**: AAC НЕ тормозит развитие речи, а поддерживает (свобода от давления "говори!")

**Context-Aware AAC** (Holyfield et al., 2025): AI-ассистированные AAC с автоматическими подсказками фраз — рост лингвистической активности и вокальных аппроксимаций.

## 2. Методика работы со слоговой структурой (русскоязычная традиция)

### 2.1 Базовые предпосылки (формируются ПЕРВЫМИ)
Согласно исследованиям, овладение слоговой структурой зависит от:
- **Фонематического восприятия** (различение звуков)
- **Артикуляционных возможностей** (мышцы лица, языка)
- **Ритмической и динамической организации движений**
- **Оптико-пространственной ориентации** (последовательная обработка)
- **Семантической стороны** (понимание значения слова)

### 2.2 Этапы коррекции (классическая методика)

**Этап 0 — Подготовительный**:
- Артикуляционная гимнастика
- Ритмические упражнения
- Развитие фонематического слуха
- Мелкая моторика

**Этап 1 — Гласные звуки**:
- Контрастные пары: А-И, А-О, У-А
- Работа над высотой, громкостью, силой голоса
- Связь с жестами (например, "окошко" — ладонь для [а], "заборчик" — кулак для [и])

**Этап 2 — Слоги**:
- Открытые слоги (СВ, СВС — согласный-гласный)
- Двусложные слова с открытым слогом: **ма-ма, па-па, во-да, ки-са** ← текущий уровень
- Послоговое проговаривание с отхлопыванием/отстукиванием
- Соединение разделённых слогов

**Этап 3 — Слоги со стечениями согласных**:
- мна-мно-мну, ста-сто-сту

**Этап 4 — Слова**:
- Последовательность по типам слоговой структуры (классификация Большаковой/Макаровой)
- Изолированно → в простой фразе

**Этап 5 — Текст**:
- Чистоговорки, скороговорки
- Загадки с послоговым проговариванием
- Связные тексты

### 2.3 Принципы работы
- **Наглядность**: картинки, схемы, кубики для слогов, символы
- **Ритмизация**: любое слоговое проговаривание связано с движением (хлопки, постукивание, прыжки)
- **Тактильность**: прощупывание шеи/горта при произнесении, тактильные карточки
- **Игровой контекст**: обучение через игру, а не "урок"
- **Послоговое проговаривание** → постепенный переход к слитному произнесению
- **Осмысление**: ребёнок должен понимать значение слова, а не только произносить

## 3. Принципы проектирования приложения

### 3.1 Научно обоснованные принципы

| Принцип | Источник | Применение в приложении |
|---------|----------|------------------------|
| **Positive Reinforcement** | ABA/VBT | Немедленное вознаграждение за попытку произнесения |
| **Shaping** | ABA | Постепенное приближение к целевому звучанию (начинаем с любой попытки) |
| **Prompting & Fading** | ABA | Подсказки, которые постепенно убираются |
| **Errorless Learning** | VBT | Минимизация ошибок через немедленные подсказки |
| **Natural Environment** | NDBI | Интеграция в игровой контекст, а не дриль |
| **Joint Attention** | NDBI | Разделение внимания между объектом и партнёром |
| **Following Child's Lead** | NDBI | Выбор контента по интересам ребёнка |
| **Multisensory** | Логопедия | Визуал + аудио + тактиль (вибрация) + моторика |
| **Rhythm & Movement** | Логопедия | Связь речи с движением |
| **AAC Support** | AAC/PECS | Картинки как опора, без давления "говори!" |

### 3.2 Дизайн-принципы для детей с РАС/ЗПРР
- **Предсказуемость**: чёткая структура, визуальный распорядок
- **Минимализм**: без лишних элементов, снижение когнитивной нагрузки
- **Визуальная поддержка**: картинки, цветовое кодирование, прогресс-бары
- **Контроль**: ребёнок выбирает, когда переходить дальше
- **Нет штрафов за ошибки**: только позитивное подкрепление
- **Короткие сессии**: 5-10 минут, с возможностью продолжить по желанию
- **Родительский дашборд**: отслеживание прогресса, настройка контента

### 4.2 Предлагаемые мини-игры

**1. "Собери слово" (слоговый конструктор)**
- Картинка + разрозненные слоги
- Ребёнок перетаскивает слоги в правильном порядке
- При правильном размещении — озвучивание + анимация
- **Навык**: понимание последовательности слогов

**2. "Хлопни в слог" (ритм-игра)**
- Слово произносится, ребёнок хлопает/тапает по экрану в такт слогам
- Визуальная пульсация картинки под каждый слог
- **Навык**: слуховое членение, ритм

**3. "Повтори за мной" (echoic training)**
- Персонаж произносит слог/слово
- Ребёнок нажимает микрофон и повторяет
- Анализ произношения (базовый: длительность, энергия; продвинутый: распознавание фонем)
- **Навык**: имитация, echoic operant

**4. "Что хочет [персонаж]?" (mand training)**
- Персонаж показывает желание (ждёт, смотрит на предмет)
- Ребёнок выбирает/произносит правильное слово
- Персонаж получает предмет и радуется
- **Навык**: функциональная коммуникация (запрос)

**5. "Найди пару" (фонематическое восприятие)**
- Карточки со слогами, нужно найти одинаковые
- С озвучиванием при переворачивании
- **Навык**: слуховое различение

**6. "Артикуляционная зарядка"**
- Анимированный персонаж показывает движения рта/языка
- Ребёнок повторяет
- **Навык**: развитие артикуляционной моторики

### 4.3 Прогрессия сложности

```
Уровень 1: CV (открытые слоги)
  → ма, па, ба, во, да, ки, са
  
Уровень 2: CV-CV (двусложные, открытые)
  → ма-ма, па-па, во-да, ки-са, мо-ло-ко
  
Уровень 3: CVC (закрытые слоги)
  → кот, дом, стол
  
Уровень 4: CCV (стечения согласных)
  → сна, мна, сто
  
Уровень 5: Смешанные многосложные
  → ма-ши-на, ко-ро-ва
```

### 4.4 Система подкрепления
- **Звёздочки/награды** за сессию
- **Анимированный персонаж** реагирует на успехи
- **"Сад/аквариум/город"** — визуальное пространство, которое растёт с прогрессом
- **Родительское уведомление** о достижениях

## 6. Критические предупреждения

1. **Приложение — дополнение, а не замена** специалистам. Регулярные занятия с логопедом и дефектологом необходимы.
2. **Никакого давления** на речь. Если ребёнок не говорит — AAC (картинки) должны быть всегда доступны.
3. **Индивидуальность**. То, что работает для одного ребёнка, может не работать для другого. Приложение должно быть гибко настраиваемым.
4. **Родительское участие** критично. Приложение должно учить РОДИТЕЛЯ, как заниматься с ребёнком.
5. **Данные о прогрессе** важны для специалистов. Экспорт статистики для логопеда.

## Приложение C. Родительский скрининг CAS (Childhood Apraxia of Speech)

> Этот скрининг — НЕ диагноз, а инструмент для родителя, чтобы лучше понять профиль ребёнка и настроить приложение. Окончательная диагностика — прерогатива логопеда/дефектолога.

### C.1 Ключевые признаки CAS (консенсусные)

Согласно исследованиям (Shriberg et al., 2011; Strand, 2017; RCSLT 2024), CAS характеризуется **тремя основными признаками**:

1. **Непоследовательность (token-to-token inconsistency)** — одно и то же слово произносится по-разному в разных попытках
2. **Нарушение коартикуляционных переходов** — трудности с плавным переходом между звуками и слогами
3. **Ошибки лексического ударения** — неправильное распределение ударения в слове (равномерное ударение на все слоги — "роботизированная" речь)

### C.2 Родительский чек-лист (адаптированный из Strand's 10-point checklist и Mayo 10)

Отметьте, что наблюдаете у ребёнка **часто** (не единичные случаи):

| № | Признак | Да | Иногда | Нет | Наблюдение |
|---|---------|-----|--------|-----|------------|
| 1 | Произношение одного и того же слова сильно варьируется от раза к разу | ☐ | ☐ | ☐ | |
| 2 | Гласные звуки "плывут" или искажаются (например, "мама" звучит как "меме" или "мому") | ☐ | ☐ | ☐ | |
| 3 | Ребёнок как будто "ищет" положение рта/языка перед произнесением слова (groping) | ☐ | ☐ | ☐ | |
| 4 | Монотонная речь или равномерное ударение на все слоги ("робот") | ☐ | ☐ | ☐ | |
| 5 | Чем длиннее слово, тем хуже произношение (мама → ок, а молоко → "моко") | ☐ | ☐ | ☐ | |
| 6 | Трудности с быстрым повторением слогов (па-па-па, та-та-та) | ☐ | ☐ | ☐ | |
| 7 | Проблемы с последовательностями типа "па-та-ка" (трудно менять положение артикуляторов) | ☐ | ☐ | ☐ | |
| 8 | Согласные звуки заменяются непредсказуемо (то "та", то "ка", то "па" для одного слова) | ☐ | ☐ | ☐ | |
| 9 | В спонтанной речи произносит хуже, чем по подражанию | ☐ | ☐ | ☐ | |
| 10 | Звучит "трудно" — ребёнок явно напрягается при произнесении, хотя мышцы в порядке | ☐ | ☐ | ☐ | |
| 11 | Проблемы с автоматическими фразами (привет, пока, спасибо — не всегда выходит) | ☐ | ☐ | ☐ | |
| 12 | Искажение гласных в слогах (вместо чётких "а-и-у" — нечто размытое) | ☐ | ☐ | ☐ | |

### C.3 Интерпретация (предварительная)

- **8+ пунктов «Да»** — высокая вероятность моторной составляющей (CAS-профиль). Рекомендуется консультация с логопедом, специализирующимся на моторных нарушениях речи. В приложении: максимальный фокус на визуализации артикуляции, мелодике, медленных переходах, тактильности.
- **4–7 пунктов «Да»** — возможен смешанный профиль (ЗПРР + элементы CAS). Приложение: баланс между классической слоговой работой и моторными техниками.
- **0–3 пункта «Да»** — вероятнее чистая ЗПРР/ОНР без значимой моторной составляющей. Приложение: классическая слоговая работа, фонематическое восприятие.

### C.4 Специализированные диагностические инструменты (для специалиста)

| Инструмент | Возраст | Что оценивает | Надёжность |
|------------|---------|--------------|------------|
| **DEMSS** (Dynamic Evaluation of Motor Speech Skill) | 3+ | Динамическая оценка моторных речевых навыков | Test-retest 89% |
| **Mayo 10 checklist** | Любой | 10 ключевых признаков CAS | Высокая чувствительность |
| **Strand's 10-point checklist** | Любой | Перцептивная оценка | 91% для 4 ключевых пунктов |
| **KSPT** (Kaufman Speech Praxis Test) | 2;6+ | Речевая праксис | Стандартизирован |
| **ReST/NDP3** | 4–12 | Артикуляция и просодика | RCT-подтверждено |

## Приложение A. Глубокое погружение: Минимально вербальные дети с нормальным слухом

> Этот раздел — для детей, которые **слышат нормально**, **могут понимать больше, чем говорят**, но их **обращённая (экспрессивная) речь** развивается крайне медленно или минимально.

### A.1 Что говорит статистика

- **25–30%** детей с РАС остаются минимально вербальными (MV) на протяжении жизни (Tager-Flusberg & Kasari, 2013; Rose et al., 2016)
- В возрасте 3 лет **50–60%** детей с РАС — МВ (Kissine et al., 2023)
- Несмотря на интенсивную терапию, **только половина МВ детей** разовьёт экспрессивную речь к 7 годам (Fountain et al., 2012; Pickles et al., 2014)
- Однако: **19–38%** МВ детей «догоняют» и становятся вербальными в подростковом/взрослом возрасте (Maltman et al., 2021; Brignell et al., 2018). Языковые траектории крайне индивидуальны.

**Критически важно**: появление экспрессивной речи связано с лучшим когнитивным развитием, адаптивным функционированием, снижением тяжести аутистичных симптомов и лучшим качеством жизни (Mouga et al., 2020).

### A.2 Рецептивный язык ≠ экспрессивный: ключевой диссонанс

Исследование Chen et al. (2023), N=1579 МВ детей с РАС (5–18 лет):

- В целом рецептивный язык у МВ детей **значимо выше** экспрессивного (на VABS-II: рецептив 7.53 vs экспрессив 5.37, p<.001)
- Но на индивидуальном уровне это касается только **24%** детей — у остальных и тот, и другой низкие
- У детей с диссоциацией (рецептив > экспрессив):
  - **Значительно лучше моторика** (t=2.51, p=.013)
  - Разрыв увеличивается с возрастом

**Вывод**: если ребёнок понимает, но не говорит — это может указывать на **моторные проблемы речи**, а не на отсутствие языковых знаний.

### A.3 Моторные нарушения речи: скрытый диагноз

**Childhood Apraxia of Speech (CAS)** — расстройство планирования/программирования речедвигательных актов:

- Chenausky et al. (2019): до **25% МВ детей с РАС** соответствуют критериям CAS
- Моторные нарушения речи **предсказывают** количество произносимых слов у МВ детей, но НЕ у LV (low verbal) — то есть у тех, кто совсем не говорит, дело может быть не только в моторике
- Maffei et al. (2023): практически все МВ дети в выборке имели нарушения точности согласных/гласных, разборчивости, координации
- Chenausky et al. (2021) через модель DIVA выделили **два профиля**:
  1. Чисто моторные проблемы (CAS)
  2. Моторные + аудиторные проблемы (CAS + APD) — эти дети плохо различают похожие звуки («мяч» vs «дядя»)

**CAS vs простая ЗПРР**: при CAS речь **непоследовательна** — сегодня ребёнок произнёс «ма-ма» идеально, завтра не может. Звуки искажены, просодика нарушена, трудности с переходами между слогами.

### A.4 Что предсказывает появление речи

| Фактор | Что предсказывает | Источник |
|--------|------------------|----------|
| **Фонетический инвентарь** (какие звуки уже произносит) | >30% вариации успеха терапии | Chenausky et al. (2018) |
| **Мелкая моторика** | Экспрессивный язык | Frontiers 2025 (SEM-модель) |
| **Невербальный IQ** | Рецептивный язык | Frontiers 2025 |
| **Социальные навыки** | Рецептивный язык (сильнейший предиктор) | Chen et al. (2023) |
| **Нарушения моторной речи** | НЕгативный предиктор экспрессивного языка | Chenausky et al. (2019) |
| **ADOS (тяжесть РАС)** | Незначительный предиктор при прочих равных | Chen et al. (2023) |

**Важно**: тяжесть аутизма (ADOS) слабо предсказывает языковые способности. Ребёнок может быть «тяжёлым» по аутизму, но иметь потенциал к речи.

### A.5 Специфические интервенции для моторной составляющей

#### PROMPT (Prompts for Restructuring Oral Muscular Phonetic Targets)
- Тактильные подсказки на лице/шее/горте для «прощупывания» правильного звука
- Sweeney & Lebersfeld (2016): 10 МВ детей с РАС + симптомами CAS, 30 сессий — 9 из 10 научились говорить хотя бы одно слово (в среднем 2.8)
- **Минусы**: требует обученного специалиста, не масштабируется в приложение напрямую, но принцип тактильности можно адаптировать

#### ReST (Rapid Syllable Transition Treatment)
- Разработан для CAS
- Работа с ненастоящими словами (pseudo-words) — снижает когнитивную нагрузку, фокус на моторном планировании
- 1 час, 4 дня/неделю, 3 недели — интенсивный формат
- RCT (Murray et al., 2015): большие эффекты, ReST лучше сохраняет результат через 4 месяца
- **Применимость к приложению**: тренировка быстрых переходов между слогами, ритм

#### NDP3 (Nuffield Dyspraxia Programme — 3rd Edition)
- «Снизу вверх»: звуки → слоги → слова → интонация
- Доказанная эффективность в RCT
- **Применимость к приложению**: последовательность уровней сложности

#### MBCT (Melodic Based Communication Therapy)
- Каждое слово — своя мелодия
- Пилотное RCT (Sandiford et al., 2013): МВ дети 5–7 лет, 5 недель — MBCT не уступает традиционной терапии
- **Применимость к приложению**: музыкальные мини-игры, мелодическое озвучивание слогов

#### AMMT (Auditory-Motor Mapping Training)
- Совмещает слуховое восприятие, моторное производство и визуальную поддержку
- Подпевая с интонацией, ребёнок одновременно слышит, видит и пытается повторить
- Эффективен для детей с хотя бы минимальными речевыми навыками

### A.6 Русскоязычная специфика

- **Скробкина (2008)**: "У детей с аутизмом сознание окружено 'грамматической глухотой'. Речь понимается приблизительно."
- **Речевой негативизм** — распространённая проблема при принуждении
- **Диссоциация пассивного и активного словаря** — понимают больше, чем произносят
- **Компьютерные программы** — особенно сильный мотиватор для детей с аутизмом (даже сильнее, чем для детей с другими нарушениями)
- **Визуальный канал** — предпочтительнее акустического у большинства

### A.7 Что это значит для приложения

#### Красные флаги — чего НЕ делать
1. **Не требовать «правильного» произношения** — при CAS звуки будут непоследовательными. Shaping: поощрять любую аппроксимацию
2. **Не давать аудио-задания без визуальной опоры** — «грамматическая глухота» + возможные АПР
3. **Не форсировать** — речевой негавизм быстро закрепляется
4. **Не игнорировать AAC** — картинки должны быть всегда доступны, даже если ребёнок уже говорит слоги

#### Зелёные флаги — что делать
1. **Тактильность**: вибрация устройства при произнесении слога, тактильные карточки
2. **Мелодика**: пение слогов, мелодические контуры (MBCT)
3. **Визуализация артикуляции**: анимированный рот, показывающий, как ставится звук
4. **Медленные переходы**: не «ма-ма» сразу, а «ма-а-а-ма-а-а» с визуальным «перетеканием»
5. **Работа с фонетическим инвентарем**: фиксировать и поощрять КАЖДЫЙ новый звук, даже если он неидеален
6. **Послоговая сегментация**: каждый слог — отдельный «блок», с визуальным разделением
7. **Минимальные пары** (для CAS+APD): "ма-на", "ба-па" — тренировка различения похожих звуков
8. **Ритм + моторика**: хлопки, постукивание, прыжки — связь речи с крупной моторикой

#### Диагностический модуль (идея)
- Ребёнок проходит 5-минутный "тест" на фонетический инвентарь
- Приложение фиксирует: какие звуки произносит, насколько последовательно, есть ли просодические нарушения
- Результат — рекомендация для родителя: "похоже на моторный профиль, рекомендуем фокус на..." (с оговоркой, что это не диагноз)

## Приложение B. Профиль целевого пользователя (анонимизированный)

> Ниже — обобщённый профиль на основе описания родителя. Используется для калибровки приложения.

### B.1 Ключевые характеристики

| Параметр | Наблюдение | Научная интерпретация |
|----------|-----------|----------------------|
| **Возраст** | 5 лет | Старший дошкольный, оптимальный возраст для интенсивной работы |
| **Диагноз** | ЗПРР, подозрение на РАС | Работаем с учётом обоих профилей |
| **Слух** | Нормальный (подтверждён сурдологом) | Проблема ЦНС, не периферическая. Исключаем сенсоневральную глухоту |
| **Интеллект** | Сохранён. Собирает большие пазлы | Хороший прогноз для рецептивного языка и когнитивных навыков |
| **Мелкая моторика** | Развита хорошо | **Сильный положительный предиктор** экспрессивного языка (Frontiers 2025) |
| **Крупная моторика** | Развита хорошо | Позволяет интегрировать моторные упражнения в речевые игры |
| **Текущий речевой уровень** | Слоговое произношение по карточкам (во-да, ки-са, ма-ма) | Уровень CV-CV, открытые слоги (Этап 2 классической методики) |

### B.2 Речевой профиль

| Аспект | Наблюдение | Интерпретация |
|--------|-----------|---------------|
| **Последовательность произношения** | «Чеканит» — стереотипно, одинаково | **Скорее ОНР 2 уровня / речевые стереотипии при РАС**: механически запомненный слоговой шаблон, воспроизводится стабильно. При CAS было бы хаотичное варьирование |
| **Инициатива в речи** | Почти всегда по запросу | **Echoic operant** преобладает, **mand** (запрос) слабо развит. Приоритет — переводить речь из «повторения» в «запрашивание» |
| **Понимание обращённой речи** | «Что-то среднее» | Рецептивный язык выше экспрессивного, но не идеальный. Работаем в обе стороны |
| **Социальная мотивация** | Работает с карточками, если структурировано | Достаточная для занятий при правильной подаче |

### B.3 Синтез: вероятный профиль

На основании данных:
- **Сохранный невербальный интеллект** + **хорошая мелкая моторика** + **нормальный слух** = отличные предпосылки для развития речи
- **«Чеканка» в произношении** (стабильные, стереотипные слоговые шаблоны) = характерно для **ОНР 2 уровня / речевых стереотипий при РАС**, а не CAS. Моторная программа работает, но **ограничена и негибкая**
- **Речь по запросу** = проблема не в знании слов, а в **инициации** речевого акта и недостаточном развитии мотивации к коммуникации
- **Понимание «среднее»** = работаем параллельно на рецептивный язык, но **фокус на экспрессивный**

**Итоговый вывод**: ребёнок находится в **зоне ближайшего развития** для экспрессивной речи. Ключевой барьер — не моторное программирование (оно работает стабильно), а **ограниченный фонетический инвентарь + отсутствие инициативной коммуникации**. Приложение должно быть заточено на:
1. **Расширение слоговых шаблонов** — новые CV-сочетания, переход к CVC
2. **Перевод echoic → mand** (из повторения в запрос) — через AAC и игровую мотивацию
3. **Разрушение стереотипии** — вариативность в произношении, гибкость
4. **AAC как мостик** — для слов, которые ещё не произносит

### B.4 Противопоказания для приложения

Приложение **НЕ должно**:
- Требовать «чистого» произношения — вариативность ожидаема и нормальна
- Исключать AAC, даже когда ребёнок уже говорит слоги
- Создавать ситуации «выиграл/проиграл» — только позитивное подкрепление
- Требовать зрительного контакта (некомфортно при РАС)
- Навязывать темп — ребёнок контролирует скорость
