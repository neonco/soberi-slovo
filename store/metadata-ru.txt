Название приложения: Собери слово
Краткое описание (до 80 символов): Логопедическое приложение для детей 5+ — собери слово из слогов
Категория: Образование / Для детей
Возрастной рейтинг: 3+
Язык интерфейса: русский

Полное описание:
«Собери слово» — развивающее приложение для дошкольников (от 5 лет),
созданное с учётом методики работы с общим недоразвитием речи (ОНР II уровень).

Ребёнок видит картинку, слышит слово диктора и собирает его из слоговых
карточек. Уровни построены по принципу от простого к сложному (по Л.Н. Марковой):
односложные слова → двусложные → трёхсложные с разным местом ударения.

Особенности:
— Крупные картинки и слоговые «пилюли», понятные ребёнку 5 лет
— Озвучка диктора (7 голосов), подсветка ударения
— Спокойная награда без перегруза внимания
— Режимы: «Собери слово», «Покажи», «Да/Нет»
— Работает как WebView-обёртка сайта (всегда актуальная версия)

Приложение не собирает персональные данные и не требует регистрации.
